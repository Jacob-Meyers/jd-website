JEY exif420

C++ Library - Created By Jeyers Development

The Simple, lightweight, & fast EXIF Reader Library for C++

JOIN THE DISCORD

Reads EXIF and Metadata from files.

Current file support:

  • .nef (Nikon Raw)
  • .tiff / .tif (Standard TIFF Images)
  • .cr2 (Canon Raw - also uses a TIFF structure base)
  • .dng (Adobe Digital Negative)
  • .pef (Pentax Raw)
  • .orf (Olympus Raw)
  • .arw (Sony Raw)
